== Description ==

Put a simple PayPal donation form on any post or page with a shortcode. The form has buttons for quick donation amounts, the option to enter your own amount, and the option to choose a one-time or monthly recurring donation.

== Installation ==

1. Install either via the WordPress.org plugin directory, or by uploading the files to your server.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Use the shortcode [vsdf_donation_form] on any page!
